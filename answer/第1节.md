## 1.2. 基本语法和数据类型

### [1] 输入一个长方体的长、宽、高，输出它的体积。

``` Python
length = float(input("请输入长方体的长："))
width = float(input("请输入长方体的宽："))
height = float(input("请输入长方体的高："))

volume = length * width * height

print("长方体的体积为：", volume)
```

### [3] 输入圆的半径，输出圆的面积和周长（圆周率可取 `3.14159`）。

``` Python
PI = 3.14159

r = float(input("请输入圆的半径："))

area = PI * r * r
perimeter = 2 * PI * r

print("圆的面积为：", area)
print("圆的周长为：", perimeter)
```

## 1.3. 循环与判断语句

[1] 输入一个整数，判断它是正数、负数还是零

``` Python
num = int(input("请输入一个整数："))

if num > 0:
    print("这是一个正数")
elif num < 0:
    print("这是一个负数")
else:
    print("这是零")
```

[2] 用 for 循环输出 1 到 100 中所有 3 的倍数

``` Python
for i in range(1, 101):
    if i % 3 == 0:
        print(i)
```

[3] 输入 5 个数，输出其中的最大值

``` Python
max_num = None

for i in range(5):
    num = float(input(f"请输入第{i+1}个数："))
    if max_num is None or num > max_num:
        max_num = num

print("最大值为：", max_num)
```

[4] 不断输入整数，直到输入 0 为止，最后输出这些非零整数的和

``` Python
total = 0

while True:
    num = int(input("请输入一个整数（输入0结束）："))
    if num == 0:
        break
    total += num

print("这些非零整数的和为：", total)
```

[5] 输出九九乘法表

``` Python
for i in range(1, 10):
    s = ""
    for j in range(1, i + 1):
        s += f"{j}×{i}={i * j} "
    print(s)
```

[6] 输入二十个大小不同的整数，请从小到大排序

排序方法存在很多种，本教程不做更高要求，下面以最为基础的选择排序法和冒泡排序法为例，其它排序方法可自行了解。

选择排序法：

``` Python
nums = []

for i in range(20):
    num = int(input(f"请输入第{i+1}个整数："))
    nums.append(num)

n = len(nums)
for i in range(n - 1):
    min_index = i
    for j in range(i + 1, n):
        if nums[j] < nums[min_index]:
            min_index = j
    nums[i], nums[min_index] = nums[min_index], nums[i]

print("从小到大排序后的结果为：")
print(nums)
```

冒泡排序法：

``` Python
nums = []

for i in range(20):
    num = int(input(f"请输入第{i+1}个整数："))
    nums.append(num)

# 冒泡排序
n = len(nums)
for i in range(n - 1):
    for j in range(n - 1 - i):
        if nums[j] > nums[j + 1]:
            nums[j], nums[j + 1] = nums[j + 1], nums[j]

print("从小到大排序后的结果为：")
print(nums)
```
