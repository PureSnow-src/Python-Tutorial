## 1.2. 基本语法和数据类型

### [1] 输入一个长方体的长、宽、高，输出它的体积。

``` Python
length = float(input("请输入长方体的长："))
width = float(input("请输入长方体的宽："))
height = float(input("请输入长方体的高："))

volume = length * width * height

print("长方体的体积为：", volume)
```

### [2] 输入两个整数，输出它们的和、差、积、商。

``` Python
a = int(input("请输入第一个整数："))
b = int(input("请输入第二个整数："))

print("和：", a + b)
print("差：", a - b)
print("积：", a * b)
print("商：", a / b)
```

### [3] 输入圆的半径，输出圆的面积和周长（圆周率可取 `3.14159`）。

``` Python
PI = 3.14159

r = float(input("请输入圆的半径："))

area = PI * r * r
perimeter = 2 * PI * r

print("圆的面积为：", area)
print("圆的周长为：", perimeter)
```

### [4] 输入姓名和年龄，输出一句自我介绍，例如：`我叫Alice，今年20岁。`

``` Python
name = input("请输入姓名：")
age = int(input("请输入年龄："))

print(f"我叫{name}，今年{age}岁。")
```

### [5] 已知列表 `scores = [78, 85, 92, 88]`，输出列表长度、第一个元素和最后一个元素。

``` Python
student = {
    "name": "Alice",
    "id": "20260001",
    "major": "Engineering"
}

print("该学生的专业是：", student["major"])
```
