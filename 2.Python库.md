# 2. Python库

在前一章中，我们学习了 Python 的基本语法、数据类型、循环、判断和函数。到了这一章，我们开始接触 Python 真正强大的地方：**库（library）**。

如果把 Python 语言本身理解为一套“语法规则”，那么各种库就像是在这套语法规则之上已经搭建好的“工具箱”。借助这些工具箱，我们不必从零开始重复造轮子，而是可以直接使用别人已经写好的功能去完成文件处理、数据分析、绘图、科学计算等任务。

Python 之所以在科研、工程、数据分析和自动化脚本等领域被广泛采用，一个很重要的原因，就是它拥有非常完整的官方标准库，以及数量庞大的第三方库生态。

本章将从五个板块介绍 Python 库的基本使用方式：

- Python 官方标准库
- 第三方库生态与管理工具
- NumPy
- Matplotlib
- Pandas

学习本章时，应当始终带着一个基本认识：

> **Python 不只是“会写语法”，更重要的是学会调用合适的库来解决问题。**

# 2.1. Python官方标准库

Python 安装完成后，就自带了一大批可以直接使用的模块，这些模块统称为 **Python 官方标准库（standard library）**。官方文档将标准库定义为随 Python 一起分发的库参考集合，其中既包括一些接近语言核心的数据类型和函数，也包括文件系统、日期时间、随机数、正则表达式、JSON 处理等常用模块。

标准库的意义在于：很多常见需求其实并不需要额外安装第三方库，直接调用标准库就能完成。例如：

- 数学运算：`math`
- 随机数：`random`
- 日期时间：`datetime`
- 文件与路径：`os`、`pathlib`
- JSON 数据处理：`json`
- CSV 文件读写：`csv`
- 正则表达式：`re`
- 命令行参数与解释器信息：`sys`
- 一些更方便的数据容器：`collections`

因此，在真正安装第三方库之前，首先应该建立一个基本观念：

> **很多基础工作，先看看标准库能不能做。**

## 模块与导入

标准库中的功能通常以“模块”的形式组织。要使用某个模块，首先需要导入它。

``` Python
import math
print(math.sqrt(16))
```

也可以只导入某个功能：

``` Python
from math import sqrt
print(sqrt(16))
```

这两种写法都可以。一般来说：

- `import 模块名` 写起来更短，但在大程序中容易和别的名字混淆；
- `from 模块名 import 名称` 更清楚，能看出函数来自哪里。

初学阶段，更推荐优先使用前一种写法。

## 数学模块：math

虽然 Python 内置了基本算术运算，但很多常见数学函数都放在 `math` 模块里。

``` Python
import math

print(math.sqrt(9))
print(math.sin(math.pi / 2))
print(math.log(10))
print(math.floor(3.8))
print(math.ceil(3.2))
```

这里：

- `sqrt()` 表示平方根；
- `sin()` 表示正弦函数；
- `log()` 表示对数；
- `floor()` 表示向下取整；
- `ceil()` 表示向上取整；
- `pi` 是圆周率常量。

对于工科和理科中的基础数值计算，`math` 是最常用的标准库模块之一。

## 随机数模块：random

`random` 模块可用于生成随机整数、随机浮点数，或者从列表中随机抽取元素。

``` Python
import random

print(random.randint(1, 10))
print(random.random())
print(random.choice(["A", "B", "C"]))
```

其中：

- `randint(a, b)` 返回区间 `[a, b]` 内的随机整数；
- `random()` 返回 `[0, 1)` 之间的随机浮点数；
- `choice()` 从序列中随机选取一个元素。

前一章中的“猜数字”程序，如果继续扩展，就可以借助 `random` 生成随机答案。

## 日期时间模块：datetime

处理时间和日期时，标准库中的 `datetime` 模块非常重要。

``` Python
from datetime import datetime

now = datetime.now()
print(now)
print(now.year)
print(now.month)
print(now.day)
```

这类功能在日志记录、实验数据命名、自动化脚本中都很常见。

如果需要把时间格式化为字符串，也可以使用：

``` Python
from datetime import datetime

now = datetime.now()
print(now.strftime("%Y-%m-%d %H:%M:%S"))
```

## 路径与文件系统：os 和 pathlib

程序经常需要读取文件、创建文件夹、判断路径是否存在。早期大家常用 `os` 模块，现在 `pathlib` 也非常常见，因为它在路径表达上更直观。

### 使用 os

``` Python
import os

print(os.getcwd())
print(os.listdir("."))
```

- `getcwd()` 获取当前工作目录；
- `listdir()` 查看某个目录中的文件和文件夹。

### 使用 pathlib

``` Python
from pathlib import Path

p = Path("test.txt")
print(p.exists())
print(p.name)
print(p.suffix)
```

`pathlib` 把路径当作对象处理，在写较复杂的文件操作程序时很方便。

## 数据交换：json 和 csv

很多程序都需要与外部文件交换数据。标准库已经提供了常见格式的处理模块。

### JSON

JSON 是一种常见的数据交换格式，常用于配置文件和网络接口返回数据。

``` Python
import json

student = {
    "name": "Alice",
    "age": 20,
    "score": 95
}

text = json.dumps(student)
print(text)
```

如果要把 JSON 字符串再变回 Python 对象：

``` Python
obj = json.loads(text)
print(obj["name"])
```

### CSV

CSV 是逗号分隔值文件，常用于表格数据交换。

``` Python
import csv

with open("scores.csv", "w", newline="", encoding="utf-8") as f:
    writer = csv.writer(f)
    writer.writerow(["name", "score"])
    writer.writerow(["Alice", 90])
    writer.writerow(["Bob", 85])
```

这说明：即使不借助 Pandas，仅用标准库也可以完成基础表格数据读写。

## 更方便的数据容器：collections

有些数据结构在内置类型中不够直接，标准库 `collections` 模块提供了更方便的工具。

例如 `Counter` 可以用来统计元素出现次数。

``` Python
from collections import Counter

text = "banana"
count = Counter(text)
print(count)
print(count["a"])
```

`defaultdict` 则可以在字典处理上减少很多判断语句。

## 标准库的学习策略

初学者面对标准库时，往往会误以为必须“全部学完”。实际上并不是这样。

更合适的学习方式是：

1. 先知道标准库里大致有哪些类别；
2. 碰到问题时，先判断是不是标准库已经能解决；
3. 学会查官方文档，而不是死记所有函数。

也就是说，标准库学习的重点不在“背”，而在于建立“遇到什么问题该去哪个模块里找工具”的意识。

## 基础算例：统计文本中每个字母出现的次数

下面用标准库中的 `collections.Counter` 写一个简单例子。

``` Python
from collections import Counter

text = input("请输入一段文本：")
count = Counter(text)

for ch, num in count.items():
    print(ch, "出现了", num, "次")
```

这个程序说明：当问题本质上已经是“计数”时，没有必要从头写一套复杂逻辑，直接调用标准库工具会更加直接。

# 2.2. 第三方库生态与管理工具

标准库虽然很强大，但它不可能覆盖所有场景。Python 真正形成生态优势的原因，是它还有非常丰富的 **第三方库（third-party packages）**。这些库大多发布在 Python Package Index（PyPI）上，用户可以按需安装。

例如：

- 数值计算：`numpy`
- 绘图：`matplotlib`
- 数据分析：`pandas`
- 网络请求：`requests`
- 机器学习：`scikit-learn`

当我们需要标准库之外的功能时，就需要借助包管理工具来安装和管理依赖。

## 为什么需要管理工具

第三方库不是“拷进文件夹就行”的简单关系。实际使用时，经常还会涉及：

- 安装库本身；
- 安装库依赖的其他库；
- 为不同项目维护不同版本；
- 创建相互隔离的运行环境；
- 确保别人也能复现同样的环境。

因此，包管理工具的作用并不只是“下载一个库”，而是帮助我们管理**依赖关系**和**运行环境**。

## pip

`pip` 是 Python 官方生态中最常用的包安装工具。pip 文档将它定义为 Python 的包安装器，可用于从 Python Package Index 以及其他索引安装包。通常建议通过 `python -m pip` 的形式调用，以明确是哪个 Python 解释器在执行 pip。

### 常见用法

``` bash
python -m pip install numpy
python -m pip install pandas
python -m pip uninstall numpy
python -m pip list
```

其中：

- `install`：安装库；
- `uninstall`：卸载库；
- `list`：查看已安装库。

### pip 的特点

- 使用广泛，是最基础的 Python 包管理工具；
- 与 PyPI 生态衔接最紧密；
- 通常与 `venv` 虚拟环境配合使用；
- 适合多数日常 Python 项目。

如果你写的是普通脚本、小型课程作业、基础数据处理程序，那么 **pip + venv** 往往已经足够。

## uv

`uv` 是 Astral 提供的 Python 工具链。官方文档将其描述为面向 Python 开发的一组核心功能集合，覆盖安装和管理 Python 版本、创建虚拟环境、管理项目依赖、运行脚本和项目等工作流。

### 常见用法

``` bash
uv venv
uv add numpy
uv add pandas
uv run python main.py
uv sync
```

### uv 的特点

- 速度很快；
- 把多个常见工作流统一在一个工具里；
- 既能管理依赖，也能处理项目环境；
- 很适合从一开始就希望把项目组织得更规范的场景。

对于新项目，尤其是希望后面继续扩展的项目，`uv` 是一个很现代的选择。

## conda

`conda` 官方文档将其定义为一个运行于 Windows、macOS 和 Linux 上的包管理与环境管理工具；更广义的说明中，conda 还强调自己并不局限于 Python，而是可用于管理多种语言的软件包与依赖。

### 常见用法

``` bash
conda create -n myenv python=3.12
conda activate myenv
conda install numpy
conda install pandas
conda list
```

### conda 的特点

- 强调环境管理；
- 不只管理 Python 包，也能管理一些底层依赖；
- 在科学计算、数据分析、跨语言依赖较多的场景很常见；
- 对包含复杂二进制依赖的库往往更友好。

在科研计算环境中，很多人会选择 Miniconda 或 Anaconda 作为起点，然后通过 `conda` 创建独立环境。

## pip、uv 与 conda 的区别

可以用一个简单的角度来理解三者：

- **pip**：最基础、最通用的 Python 包安装工具；
- **uv**：更现代、更统一、更高效的 Python 项目与依赖管理工具；
- **conda**：更强调环境管理，也更擅长处理一些复杂依赖。

对于初学者，可以先记住下面这条经验：

- 普通 Python 学习项目：优先考虑 `pip` 或 `uv`；
- 科学计算、复杂依赖环境：可以考虑 `conda`；
- 无论使用哪种工具，都应尽量避免把所有库都装在同一个全局环境里。

# 2.3. NumPy

`NumPy` 是 Python 科学计算生态的基础库之一。它最核心的数据结构是 **ndarray**，即多维数组对象。与 Python 内置列表相比，NumPy 数组在数值计算中通常更高效，也更适合进行批量运算。

在前一章中，我们已经学过列表。但列表更强调“通用容器”，可以混合保存不同类型的数据；而 NumPy 数组更强调“规则的数值数据”。

## 创建数组

``` Python
import numpy as np

arr = np.array([1, 2, 3, 4])
print(arr)
print(arr.shape)
print(arr.dtype)
```

这里：

- `shape` 表示数组形状；
- `dtype` 表示元素类型。

## 基本运算

NumPy 的一个重要特点是支持向量化运算。

``` Python
import numpy as np

arr = np.array([1, 2, 3, 4])
print(arr + 1)
print(arr * 2)
print(arr ** 2)
```

这些运算会自动作用到数组中的每个元素。

## 二维数组

``` Python
import numpy as np

A = np.array([
    [1, 2, 3],
    [4, 5, 6]
])

print(A.shape)
print(A[0, 1])
print(A[:, 1])
```

二维数组在矩阵运算、网格数据处理、图像处理等场景中很常见。

## 常用函数

``` Python
import numpy as np

arr = np.array([1, 2, 3, 4, 5])
print(np.sum(arr))
print(np.mean(arr))
print(np.max(arr))
print(np.min(arr))
```

## 基础算例：计算一组测量值的平均值与标准差

``` Python
import numpy as np

values = np.array([9.8, 10.1, 9.9, 10.0, 10.2])

print("平均值：", np.mean(values))
print("标准差：", np.std(values))
```

对于实验数据、数值模拟结果、批量测量值处理，这类操作非常常见。

# 2.4. Matplotlib 和 pandas

## 2.4.1. Matplotlib

`Matplotlib` 是 Python 中最常用的绘图库之一。它可以绘制折线图、散点图、柱状图、直方图等多种图形，是数据可视化的基础工具。通过下述命令完成安装：

``` Bash
pip install matplotlib
```

对于初学者来说，最先掌握的通常是折线图。

## 最基础的折线图

``` Python
import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [1, 4, 9, 16, 25]

plt.plot(x, y)
plt.show()
```

这段代码表示：

- 用 `plot()` 画图；
- 用 `show()` 显示图像。

## 添加标题与坐标轴标签

``` Python
import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [1, 4, 9, 16, 25]

plt.plot(x, y)
plt.title("y = x^2")
plt.xlabel("x")
plt.ylabel("y")
plt.grid(True)
plt.show()
```

图像不仅要能画出来，还要尽量具备清晰的标题、坐标轴说明和网格，这样才更适合用于分析和展示。

## 基础算例：绘制正弦函数曲线

``` Python
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(0, 2 * np.pi, 200)
y = np.sin(x)

plt.plot(x, y)
plt.title("sin(x)")
plt.xlabel("x")
plt.ylabel("sin(x)")
plt.grid(True)
plt.show()
```

这个例子结合了 NumPy 与 Matplotlib：NumPy 负责生成数据，Matplotlib 负责把数据画出来。

## 散点图

折线图强调数据点之间的连续变化关系，而**散点图（scatter plot）**更适合展示两个变量之间的离散对应关系。

例如，在实验数据处理中，我们经常会记录一组自变量与因变量的测量值。若希望先观察这些数据点的大致分布情况，而不是直接用线连接它们，就可以使用散点图。

``` Python
import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5, 6]
y = [2.1, 3.9, 6.2, 7.8, 10.1, 12.2]

plt.scatter(x, y)
plt.title("Scatter Plot")
plt.xlabel("x")
plt.ylabel("y")
plt.grid(True)
plt.show()
```

这里：

- `scatter()` 用于绘制散点图；
- `x` 和 `y` 中的一对数据确定图中的一个点；
- 散点图常用于观察数据是否近似线性、是否存在异常点，以及变量之间的大致关系。

如果后续想进一步做曲线拟合、线性回归或误差分析，散点图通常是很好的第一步。

## 基础算例：绘制实验测量数据的散点图

下面给出一个简单例子。假设某次实验中记录了位移与载荷的对应数据，我们先用散点图观察它们之间的关系。

``` Python
import matplotlib.pyplot as plt

x = [0, 1, 2, 3, 4, 5]
y = [0.0, 1.1, 2.0, 3.2, 4.1, 5.3]

plt.scatter(x, y)
plt.title("Load-Displacement Data")
plt.xlabel("Displacement")
plt.ylabel("Load")
plt.grid(True)
plt.show()
```

这类图像在实验数据的初步整理中很常见，因为它能够直接显示每个测量点的位置。

## 柱状图

**柱状图（bar chart）**常用于比较不同类别数据的大小。例如比较不同学生成绩、不同组实验结果、不同材料参数等。

``` Python
import matplotlib.pyplot as plt

names = ["A", "B", "C", "D"]
scores = [82, 91, 76, 88]

plt.bar(names, scores)
plt.title("Scores of Students")
plt.xlabel("Student")
plt.ylabel("Score")
plt.grid(True, axis="y")
plt.show()
```

这里：

- `bar()` 用于绘制柱状图；
- 横坐标通常表示类别；
- 纵坐标表示对应类别的数值大小；
- 在柱状图中，常用 `axis="y"` 只显示水平方向的网格，便于读取柱子的高度。

柱状图不强调数据点之间的连续变化，而是强调不同对象之间的数量比较。

## 基础算例：绘制不同试件的抗压强度柱状图

假设有四个试件，测得它们的抗压强度如下。现在用柱状图对比它们的大小。

``` Python
import matplotlib.pyplot as plt

samples = ["Specimen 1", "Specimen 2", "Specimen 3", "Specimen 4"]
strength = [35.2, 37.8, 36.5, 38.1]

plt.bar(samples, strength)
plt.title("Compressive Strength of Specimens")
plt.xlabel("Specimen")
plt.ylabel("Strength (MPa)")
plt.grid(True, axis="y")
plt.show()
```

这个例子说明：当数据本身属于“不同对象的数值比较”时，柱状图通常比折线图更合适。

## 2.4.2 Pandas：表格数据处理工具

如果说 `NumPy` 更适合规则的数值数组，那么 `pandas` 更适合处理带有“行、列、表头”概念的数据，例如实验记录表、成绩表、财务表、传感器采样结果等。

`pandas` 中最常见的两个对象是：

- `Series`：带索引的一维数据；
- `DataFrame`：二维表格数据结构。

实际使用中，最常接触的是 `DataFrame`。

### 导入方式

``` Python
import pandas as pd
```

和 `numpy as np` 类似，`pd` 也是非常常见的简写方式。

### 创建 Series

``` Python
import pandas as pd

s = pd.Series([10, 20, 30, 40])
print(s)
```

如果给它添加索引：

``` Python
import pandas as pd

s = pd.Series([10, 20, 30], index=["A", "B", "C"])
print(s)
print(s["B"])
```

这说明 `Series` 不只是一个简单列表，它还可以用索引标签来访问数据。

### 创建 DataFrame

最常见的方式是用字典创建表格：

``` Python
import pandas as pd

data = {
    "name": ["Alice", "Bob", "Charlie"],
    "age": [20, 21, 19],
    "score": [88, 92, 85]
}

df = pd.DataFrame(data)
print(df)
```

这里：

- 每个键对应一列；
- 每列中的列表长度应当一致；
- `DataFrame` 的输出形式非常像一张表。

### 查看基本信息

``` Python
import pandas as pd

data = {
    "name": ["Alice", "Bob", "Charlie"],
    "age": [20, 21, 19],
    "score": [88, 92, 85]
}

df = pd.DataFrame(data)

print(df.shape)
print(df.columns)
print(df.head())
```

其中：

- `shape` 返回表格的行数和列数；
- `columns` 返回列名；
- `head()` 默认显示前 5 行数据。

在处理大表格时，`head()` 是最常用的快速查看方法之一。

### 选取列与行

从 `DataFrame` 中选取一列：

``` Python
print(df["score"])
```

选取多列：

``` Python
print(df[["name", "score"]])
```

使用 `iloc` 按位置选取行：

``` Python
print(df.iloc[0])
print(df.iloc[0:2])
```

使用 `loc` 按标签选取时，更适合那些已经设置了明确行索引的表格。初学阶段，可以先把 `iloc` 理解为“按行号和列号访问”。

### 数据统计

`pandas` 对表格数据的统计非常方便。例如：

``` Python
import pandas as pd

data = {
    "score": [88, 92, 85, 90, 95]
}

df = pd.DataFrame(data)

print(df["score"].mean())
print(df["score"].max())
print(df["score"].min())
print(df["score"].sum())
```

如果想一次性查看某列的整体统计信息，可以使用：

``` Python
print(df["score"].describe())
```

它会给出计数、平均值、标准差、最小值、分位数和最大值等信息。

### 条件筛选

表格处理中最常见的工作之一，就是按条件筛选数据。

``` Python
import pandas as pd

data = {
    "name": ["Alice", "Bob", "Charlie", "David"],
    "score": [88, 92, 85, 96]
}

df = pd.DataFrame(data)

high_score = df[df["score"] >= 90]
print(high_score)
```

这个例子表示：从原表中选出成绩大于等于 90 的所有行。

### 读写 CSV 文件

现实中的数据往往不是直接写在程序里，而是保存在文件中。对于表格数据，最常见的格式之一就是 CSV。

``` Python
import pandas as pd

df = pd.read_csv("scores.csv")
print(df.head())
```

把处理后的表格重新保存为 CSV 文件：

``` Python
df.to_csv("new_scores.csv", index=False)
```

其中 `index=False` 表示保存时不额外写入行索引。

### 缺失值的基本处理

真实数据中经常会出现缺失值。`pandas` 提供了基本的判断和处理方法。

``` Python
import pandas as pd

data = {
    "name": ["A", "B", "C"],
    "score": [90, None, 85]
}

df = pd.DataFrame(data)

print(df.isnull())
print(df["score"].mean())
print(df.fillna(0))
```

这里：

- `isnull()` 用于判断哪些位置是缺失值；
- `fillna(0)` 用 0 填充缺失值。

对于初学者来说，先知道“数据不一定总是完整的”就已经很重要。
