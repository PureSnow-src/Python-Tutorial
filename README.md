# Python-Tutorial

A self-authored Python tutorial designed for beginners with no programming experience who want to dive right into Python project development. The goal of this tutorial is to help you quickly master the basics of Python within two weeks, enabling you to understand basic Python code and start coding on your own.

Currently, only the Chinese version is being written.

目前尚在编写中文版，力求快速简洁且不复杂。教程针对于准备投入Python开发工作的新人，他们可能缺乏基本的编程基础和经验，也难以理解复杂的数据结构和算法。

本教程不指望新人能快速掌握充足的代码编写能力和素养，而是希望一个没有编写代码的新人能够读懂或使用AI理解基本代码，并快速融入到涉及Python代码开发的小组中。
